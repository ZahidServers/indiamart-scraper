Changelog

All notable changes to IndiaMart Lead Scraper are documented here.

v7.0 June 2026 — Latest

Major rewrite — SPA mode & full UI overhaul

Added

  • Full SPA (Single Page Application) mode — the scraper now detects IndiaMart's React-based DOM updates in-place without requiring any page reloads
  • Pause & Resume — stop mid-scrape and continue from exactly where you left off; all collected leads are preserved
  • Auto-resume on refresh — if the page unexpectedly reloads during a scrape, the scraper automatically picks up from storage
  • Deduplication via row ID tracking — leads are never double-counted across multiple scraping sessions or page navigations
  • Real-time progress bar with smooth animated gradient fill
  • Three-stat dashboard showing Collected / Total / Progress % at a glance
  • Status badge with four distinct states: Idle, Scraping, Paused, Complete
  • Live scrolling activity log showing per-page progress
  • Excel (XLSX) export using bundled SheetJS — auto-sized column widths, no external dependency
  • CSV export with UTF-8 BOM for correct character encoding when opened in Excel
  • Graceful XLSX → CSV fallback if Excel generation fails
  • Download buttons are always available once any leads are collected — not locked behind completion
  • "Clear & Reset" button with confirmation dialog to prevent accidental data loss
  • About tab with Webaon services showcase and call-to-action
  • Complete dark UI with Inter font, gradient accents, and smooth hover transitions
  • Manifest V3 compliance

Changed

  • Complete rewrite of the scraping engine from page-reload-based navigation to SPA-aware DOM polling with waitForRowChange()
  • Storage architecture unified under a single ims key in chrome.storage.local for reliability and atomic updates
  • UI rebuilt entirely from scratch with improved layout, typography, iconography, and button state management
  • CSV export moved to the popup directly (reads from storage) — always works even when the IndiaMart tab is closed
  • Polling interval reduced to 800ms for snappier UI updates
v6.x and earlier Legacy

Page-reload based scraper

Legacy versions relied on full page reloads to navigate between pages of the Lead Manager. These versions are no longer compatible with IndiaMart's current React SPA architecture and are not supported. Upgrade to v7.0 for reliable scraping.
